# ITV-Style Scrolling Navigation (Elementor Plugin)

A custom Elementor-based navigation widget inspired by ITV-style horizontal scrolling menus.  
This plugin provides a smooth, mobile-friendly way to display navigation items in a scrollable format, improving usability for content-heavy sections.

---

## 🚀 Features

- **Horizontal Scrolling Navigation**  
  Display menu items in a smooth, scrollable row without using dropdowns.

- **Elementor Widget Integration**  
  Fully integrated as a native Elementor widget with drag-and-drop support.

- **Menu & Dynamic Content Support**  
  Use manual items or pull directly from WordPress menus.

- **Mobile-First Design**  
  Optimised for touch interaction with smooth horizontal scrolling.

- **Customisable Styling**  
  Control spacing, typography, alignment, and hover effects within Elementor.

---

## ⚙️ How It Works

1. **Add Widget in Elementor**  
   Drag the scrolling navigation widget into your layout.

2. **Configure Content Source**  
   Choose between manual items or a WordPress menu.

3. **Enable Horizontal Scroll**  
   Items are displayed in a row with overflow scrolling.

4. **Responsive Behaviour**  
   Automatically adapts to mobile, tablet, and desktop layouts.

5. **Style via Elementor Controls**  
   Customise design directly in Elementor without writing code.

---

## 📦 Installation

1. Upload the plugin ZIP via WordPress:
   - Plugins → Add New → Upload Plugin
2. Activate the plugin
3. Open Elementor editor
4. Search for the widget and add it to your page
